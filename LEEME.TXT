Formulario Productos
=====================================

REQUISITOS DEL SISTEMA:
- PHP 7.4 o superior
- PostgreSQL 12 o superior

PASOS DE INSTALACIÓN:
====================

1. CONFIGURAR BASE DE DATOS:
  - Crear una base de datos PostgreSQL llamada "sistema_productos"
  - Ejecutar el archivo sql/database.sql para crear las tablas y datos de prueba
  - Comando: psql -U postgres -d sistema_productos -f sql/database.sql

2. CONFIGURAR CONEXIÓN:
  - Abrir el archivo config/database.php
  - Modificar los datos de conexión:
    * $host: dirección del servidor PostgreSQL (por defecto: localhost)
    * $username: usuario de PostgreSQL (por defecto: postgres)
    * $password: contraseña del usuario
    * $port: puerto de PostgreSQL (por defecto: 5432)

3. CONFIGURAR SERVIDOR WEB:
   
   OPCIÓN A - Servidor integrado de PHP:
   - Abrir terminal en la carpeta del proyecto
   - Ejecutar: php -S localhost:8000
   - Acceder a: http://localhost:8000

   OPCIÓN B - XAMPP/WAMP:
   - Copiar el proyecto en la carpeta htdocs (XAMPP) o www (WAMP)
   - Acceder a: http://localhost/sistema-productos

4. VERIFICAR FUNCIONAMIENTO:
   - Acceder a la URL configurada
   - Verificar que el formulario se carga correctamente
   - Probar el registro de un producto
